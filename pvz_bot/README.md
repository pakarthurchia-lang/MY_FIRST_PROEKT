# AI Director PVZ — Telegram Bot

Telegram-бот для владельца ПВЗ Ozon. Автоматизация аналитики, учёта прибыли и управления претензиями.

## Текущее состояние

### ✅ Работает
- **📋 Претензии** — получение активных претензий из Ozon API, статусы, суммы, дедлайны, алерты за 24ч и 2ч
- **💰 Прибыль** — выбор месяца, разбивка по ПВЗ: вознаграждение (из PDF) − налог − штрафы
- **📊 Аналитика** — принятые посылки, уникальные клиенты, частота заказов, рейтинг по каждой ПВЗ
- **📝 Оборот** — ручной ввод товарооборота по ПВЗ и месяцу

### ⚠️ Не решено
- **Авторизация** — токен Ozon вставляется вручную в `data/ozon_token.json` раз в ~3 дня. Автоматического обновления через Safari нет (ограничение macOS на чтение cookies)

## Запуск

```bash
cd pvz_bot
source venv/bin/activate
python main.py
```

Скопируй `.env.example` в `.env` и заполни токены перед первым запуском.

## Стек

Python 3.9 · aiogram 3.x · aiohttp · aiosqlite · pdfplumber · APScheduler
